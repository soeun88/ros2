# ROS2

### 프로젝트:
- ROS Node 구현 

### 목표:
-  ROS(Robot Operating System)을 활용하여 Node 간 통신을 구현
  
### 주요 업무:
- ROS Node 생성
- topic 발행/구독 과정 구현
- rqt_graph를 사용해 데이터 흐름 시각화 및 관리


### 사용 기술:
- ubuntu
- ros
- c++
- rqt
